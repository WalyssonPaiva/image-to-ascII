<h2>:information_source: Como usar: </h2>
<p> ps: você precisa ter o Ruby instalado </p>
Em seu terminal:

``` 
Clone this repository
$ git clone https://github.com/WalyssonPaiva/image-to-ascII.git

Go into the repository
$ cd image-to-ascII

Install dependencies
$ bundle install

Run
$ ruby src/main.rb

```
